Understanding Search Methods

Hybrid Search

Best of both worlds! Combines keyword matching with AI-powered semantic understanding.

Finds exact keyword matches
Understands meaning and context
Handles synonyms and related concepts
Most comprehensive results
Example: Searching "AI marketing" finds content about "artificial intelligence advertising" and "machine learning promotion"

Keyword Search

Precise matching! Uses advanced text search algorithms to find exact words and phrases.

Finds exact word matches
Handles phrase searches
Supports wildcards and operators
Fast and reliable
Example: Searching "social media" finds exact matches for "social media" but not "social networking"

Semantic Search

AI-powered understanding! Uses machine learning to understand meaning and find related concepts.

Understands context and meaning
Finds conceptually similar content
Works with different languages
Great for exploratory searches
Example: Searching "customer satisfaction" finds content about "client happiness" and "user experience"

When to Use Each Method

Use Hybrid when:
  • • You want the best results
  • • You're not sure which method to use
  • • You need both precision and recall
  • • You're doing complex searches
Use Keyword when:
  • • You know exact terms to search
  • • You need fast, precise results
  • • You're searching for specific phrases
  • • You want to avoid false positives
Use Semantic when:
  • • You want to explore related topics
  • • You're not sure of exact terms
  • • You want to discover new content
  • • You're doing research or brainstorming
